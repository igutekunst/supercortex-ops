## Developer Journal
We maintain a rigorous "Dev Journal" to track design decisions and lessons learned.
*   **Location**: `docs/dev-journal/`
*   **Format**: `NNNN-descriptive-title.md`
*   **Process**:
    1.  For every significant change or decision, create a new entry.
    2.  Keep it concise: Key decisions, configs, mistakes, lessons.
    3.  Update `docs/dev-journal/index.md` with a link and summary.
